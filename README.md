## Clone this repo

1. Clone :

```sh
  git clone https://github.com/Augustob790/Experweb_test.git
```

2. Go to folder:

```sh
  cd experweb_app
```

3. Execute in terminal:

```sh
  flutter pub get && flutter run      
```
